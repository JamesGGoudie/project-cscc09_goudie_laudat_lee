Controls

"G" = Grab/move | "S" = scale | R = rotate

"+/-" = increase/decrease size of control

"Shift" = turn on snap to grid | "Ctrl" = turn off snap to grid

"Left-click and drag" = orbit | "Right-click and drag" = pan

"Scroll" = zoom in/out | "C" = reset camera

"Esc" = deselect current object | "Backspace" = delete current object
